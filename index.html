As a foodie, I used to search better restaurants for different kinds of foods on Yelp, Open table or Grubhub, while the choices based on these websites were often below my expectations. One day, I got a idea why not learn and use machine learning to help me find some restaurants which can give me surprise instead. Then I took several online courses and developed a cross-platfrom application, designed by content-base recommendation algorithm, to mark favor levels for different restaurants according to users' choices. More friends invited to use, more luckily I could find my favorite. Through this project, I realized even small widgits with machine learning could still help people make much better decisions in different areas. This experience inpired me in implementing different machine learning models and libraries to create applications for more accurate decision making.
